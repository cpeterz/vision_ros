# model

识别模型库

## 使用方法

***非常重要！！！请严格遵守***

1. master 分支为空白分支，请在这里更新版本号；除 master 以外的所有分支都将被设为只读分支，不允许进行推送
2. 如果需要更新模型，请从 master 分支新建新分支。所有分支以版本号命名，如 v1.0

   > 需满足正则表达式 `v[0-9]*[.][0-9]*`
3. 在 Armor 仓库的 CMakeLists.txt 中声明了所使用到的 model 版本，需与该仓库的版本（分支名）对应，否则 CMake 将会报错
